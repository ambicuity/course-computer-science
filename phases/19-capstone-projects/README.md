# Phase 19 — Capstone Projects

Multi-week builds that combine systems, theory, and software engineering into real artifacts.

- Lessons: 18
- Estimated time: ~219 hours
- Outcome: a substantial shipped build and final capstone package

## Tracks

Phase 19 includes concrete build tracks (compiler, kernel, database, networking, graphics, ML, security, verification) followed by an open capstone.

## Suggested completion order

1. Pick one deep systems track aligned with your goals.
2. Complete at least one verification-heavy track (testing, formal methods, static analysis).
3. Use lesson 18 to package a learner-defined project with mentor review.

## Deliverables for phase completion

- At least one fully implemented major build from lessons 01–17.
- One open capstone package from lesson 18.
- Documentation bundle: architecture, testing evidence, tradeoffs, and demo instructions.
