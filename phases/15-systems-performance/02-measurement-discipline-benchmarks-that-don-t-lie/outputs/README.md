# Outputs — Measurement Discipline

This directory contains the reusable artifact for Lesson 15.02:

**`benchmark_checklist.md`** — A one-page reference card covering:
- Common benchmark pitfalls and how to prevent each one
- When to use median vs mean vs p99 vs min
- What production frameworks (Google Benchmark, Criterion) provide
- Confidence interval formula and practical sample-size rule
- Outlier removal policy
- Pre-flight checklist before publishing benchmark numbers

Print it, pin it to your monitor, and check every box before you publish a benchmark number.
