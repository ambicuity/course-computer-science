# Output Artifact: Test Portfolio Decision Checklist

This lesson ships a reusable checklist for selecting unit/integration/E2E mix.

## Checklist

1. Define top failure risks.
2. Map each risk to primary test level.
3. Estimate runtime and flake budget.
4. Cap E2E to business-critical journeys.
5. Track escaped defects and rebalance quarterly.

## Reuse

Use this checklist when creating new services, rebuilding CI strategy, or
recovering from E2E suite bloat.
