# Code of Conduct

This project is an open learning space. Keep participation focused, respectful, and useful.

## Expected Behavior

- Be direct without being dismissive.
- Critique ideas, code, explanations, and tests rather than people.
- Assume learners arrive with different backgrounds and constraints.
- Keep issues and pull requests on topic.
- Credit sources when adding explanations, examples, or references.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Discriminatory language or conduct.
- Publishing private information without consent.
- Repeated off-topic comments after being asked to stop.
- Deliberately misleading learners with known-bad technical advice.

## Reporting

Report conduct concerns through the repository issue tracker if public discussion is appropriate. For private reports, contact the maintainer listed in `README.md`.

Reports should include what happened, where it happened, and any relevant links or screenshots. The maintainer may remove comments, close issues, block users, or take other reasonable moderation steps to keep the project usable.

## Scope

This code of conduct applies to project spaces, including issues, pull requests, discussions, and community channels connected to this repository.
